Howdy, Simon! Here's the class where you, yes, YOU get to make the robit go brrrrrrrrr. The legalDos list is important. It's all of the actions you can take. If the action only has 1 number, like TURN, LIFT, or CAROUSEL, just use x. If the action has 0 numbers, like BUCKET, ignore both x and y. If you set a WAIT time, the robot will start doing the thing and then go on to the next thing after the WAIT is over, REGARDLESS OF IF THE THING ACTUALLY FINISHED.
